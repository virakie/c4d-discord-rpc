Discord Rich Presence plugin for Cinema 4D. Shows scene name, C4D version, and elapsed time on your Discord profile.

## Installation
1. Download and unzip `c4d_discord_rpc.zip`
2. Drop the `c4d_discord_rpc` folder into your C4D plugins directory
3. Restart Cinema 4D
4. Access via Extensions > Discord RPC Settings
